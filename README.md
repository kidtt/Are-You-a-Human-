# Are-You-a-Human-

A tiny browser quiz that attempts to decide if the user is human or a robot.

How to use
- Open `index.html` in a browser.
- Click the **Start Test** button and answer the prompts.
- The result will appear under the button and also in the browser console.

Notes
- The main file was renamed from `idex.html` to `index.html` to fix a filename typo.
